Google Earth Typewriter


Why I've done it

During the first ever Google DevFest Portugal we had to come up with an idea (using Google technology), implement it in 24h and present it. I decided to use Google Maps API and some responsive web technology presented during the event.


Lessons learned

The website is uses Twitter Bootstrap, Fittext, Google Maps API, and Soundjay's typewriter sounds (besides Javascript and jQuery).
